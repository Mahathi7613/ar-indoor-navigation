Indoor Navigation AR Project

Overview
This is an Indoor Navigation system built using Augmented Reality (AR) in Unity.  
It allows users to navigate indoor spaces in real-time using AR markers and provides an interactive spatial mapping experience.


Features
- Marker-based AR navigation for indoor environments  
- Real-time spatial mapping and guidance  
- Interactive UI for user-friendly navigation  
- Scalable design for different building layouts



Technologies Used
- Unity: Game engine for AR development  
- C#: Scripting in Unity  
- AR Foundation / ARKit / ARCore: AR frameworks  
- Git: Version control



Folder Structure
- `Assets/` – Project assets and scripts  
- `Packages/` – Unity packages  
- `ProjectSettings/` – Unity project settings  
- `IndoorMap.sln` – Solution file for reference  

> Ignored folders: `Library/`, `Logs/`, `UserSettings/`, `.vscode/`
